# StundenCheck

## Development

Prerequisites:

- PHP 8.0 or later
- Composer

```bash
git clone https://github.com/devmount/stunden-check
cd stunden-check
composer install
touch database/database.sqlite
php artisan migrate
php artisan serve
```
